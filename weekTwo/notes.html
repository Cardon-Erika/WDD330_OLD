Week Two Notes - Novice to Ninja

Programming Basics

  • Use const when the variable be reassigned to another value.
  • Use let if the variable might be reassigned later in the program.
  • var not really used anymore.
  • ANSWER different from Answer as well as answer.
  • .length used to find the length of a string. Helpful with loops.
  • Template literals are created with the backtick character `. You can use both types of quote marks inside. It also allows for interpolation of JavaScript code, so a JS expression can be inserted inside the string and the result will be displayed. IE, you can call variables into strings easily using ${}.
  • 8 is different than '8'.

Arrays, Logic, and Loops

  • Remember, arrays start with 0.
  • You can use .length to find the amount of items in an array.
  • Use pop() method to remove the last item from an array.
  • Use shift() method to remove the first item from an array.
  • Use push() method to append a new value to the end of the array.
  • Use unshift() method to append a new value to the beginning of the array.
  • sort() does not work the same for numbers. [1, 3, 12, 23, 41, 9] would list as [1, 12 ,23, 3, 41, 9].
  • Use a switch statement to make a long list of if/else statements easier to test and read.

Functions

  • Functions reduce repetition. Yea!
  • All functions need to be invoked to run. To do this type its name followed by parentheses.
  • Keep your code DRY - Don't Repeat Yourself.
  • Can declare a function with arrow syntax with ES6. const square = x => x*x;
  • forEach() can be used in place of a for loop to loop through each value in an array.
  • reduce() iterates over each value in an array and cumulatively combines each result to return just a single value. Can use to sum an array of numbers. This can also be paired with other methods to improve their usability, such as map() and mean().
