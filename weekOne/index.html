Master Mobile UX

Mobile First

  • "If you design for the smallest, least powerful device first, then you will focus on the content, and your core functionality." Joel Marsh
  • Desktop was the only screen to develop for for 30+ years. In 2007, with the iPhone, suddenly new devices. Now, there are many different devices and screen sizes and resolutions.
  • With mobile devices household have, on average, 2-3 devices per person.
  • Just because your website is responsive doesn't mean it's mobile first.
Things to Remember
  1. Desktop computer sales have been declining since 2011.
  2. You don't see people with their heads in laptops.
  3. Start small and scale up.
  4. Mobile is not a fad - we aren't suddenly going to get rid of our phones.

Understanding Users

  • Be there
  • Be Useful
  • Be Quick
How Do Users Handle Device - Reachabliity Matters
  • What can the user reach with the most common grips?
  • Reduce image usage
  • SVG/Webfonts are lower bandwidth
  • Optimize and minimize CSS and JS files

Strategic Menu Design

In a typical design, the nav menu is in one of the hardest to reach areas of the screen with a one-handed grip. Simply moving the nav menu to the bottom of the screen makes it easy to reach.

Mobile Friendly Forms

  • One easy option - ditch labels and use placeholder text instead. Also, float labels help make it concise.
  • Large check boxes that are easy to hit with the thumb.
  • Simplify, and cut out what isn't needed.
  • Enable autocomplete.
  • Unless you are in a very public place, people around you won't be able to see your passwords as you type them in. Set them to display by default, and allow users to hide if desired.
  • Add spacing to credit card numbers and autoadvance to next field.
  • Use the best keyboard for what the user is entering.
  • Use anchor tag to link phone number so users can easily make a call.

Mobile Advertising

  • If you are placing ads in mobile content, make it small.
  • Stop using pop-ups and popovers.
  • Content first - users don't visit your site to look at ads.
